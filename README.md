<h2 id="description">📑 Descrição do projeto</h2>

Jogo no qual o usuário tentará adivinhar um número aleatório entre 1 e 20. Quanto menos tentativas forem necessárias, maior a pontuação.

O objetivo é explorar conceitos de **manipulação de DOM** (JavaScript), utilização de **funções semânticas** e garantir uma boa UI, com uso de CSS.  

### Funcionalidades:

- Comparar o número digitado com o número aleatório gerado pelo programa;
- Apresentar dica ao usuário sobre o número digitado;
- Mudar a aparência da página caso o usuário vença;
- Botão para reiniciar o game;

<h2 id="stacks">👨‍💻 Tecnologias</h2>

**Front-end:** HTML, CSS e JavaScript (vanilla).
